# STS2-Optional-Rock-Paper-Scissors

一个用于《Slay the Spire 2》多人共享遗物选择场景的可选石头剪刀布模组。

当多名玩家选择同一个遗物时，原版通常会进入自动判定流程。这个模组会拦截该流程，改为由玩家手动进行石头、剪刀、布决定归属，并在界面上展示当前选择状态与回合历史。

## 功能

- 拦截共享遗物选择中的自动冲突结算
- 将同一遗物的争夺改为手动石头剪刀布
- 支持键盘快捷输入
- 支持确认与取消当前选择
- 支持多回合淘汰制结算
- 显示玩家手势标记与结果历史面板
- 在结算完成后继续原始遗物发放流程

## 当前交互方式

在共享遗物选择界面中：

- 按 `1` 选择 `Rock`
- 按 `2` 选择 `Paper`
- 按 `3` 选择 `Scissors`
- 点击 `Confirm` 提交当前选择
- 再次确认已提交的同一选择时，可取消当前提交

当所有参与争夺的玩家都提交后，模组会自动推进该轮结果。
如果出现平局或未决胜情况，模组会进入下一轮，直到决出唯一胜者。

## 适用场景

这个模组主要针对多人模式下的共享遗物选择冲突：

- 两名或多名玩家选择了同一个遗物
- 你希望用手动石头剪刀布代替自动判定
- 你希望在 UI 中清楚看到每轮对局结果

## 项目结构

- `Services/`：核心流程与石头剪刀布协调逻辑
- `Patches/`：Harmony 补丁，负责接入游戏原有流程
- `Ui/`：模组内的界面与按钮逻辑
- `RoundHistory/`：回合历史显示
- `Rock-godot/`：Godot 侧资源与 `.pck`
- `Models/`：对局、回合与状态模型
- `Protocol/`：玩家选择编码与同步
